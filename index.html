Toggle usage

<button class="btn btn-primary material-button" id="myButton" data-material="cross|arrow-left|arrow-right|triangle|rect|dial">
<span class="material-button-item material-button-item-1"></span>
<span class="material-button-item material-button-item-2"></span>
<span class="material-button-item material-button-item-3"></span>
</button>
<script src="http://code.jquery.com/jquery-1.11.1.min.js"></script>
<script type="text/javascript" src="js/jquery.materialbutton.js"></script>
<script type="text/javascript">
$('#myButton').on('material.activated', function()
{
    console.log("activated!");
}).on('material.inactivated', function()
{
    console.log("inactivated!");
}).materialbutton();
</script>

Toggle events

Events Description
material.activated This event fired on activated.
material.inactivated This event fired on inactivated.
